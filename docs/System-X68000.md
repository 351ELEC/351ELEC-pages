# Sharp X68000

## Overview

- Core: `px68k` (default)
- Path: `x68000`
- Supported Extensions: `.dim .img .d88 .88d .hdm .dup .2hd .xdf .hdf .cmd .m3u .zip .7z	`
- Bios: [Required + Optional](#bios)
- Documentation: [PX68k](https://docs.libretro.com/library/px68k/)
